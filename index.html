Here are some of the projects I have done in the past related to programming.

MNLTour

MNLTour is a virtual tour system for selected tourist spots situated within the city of Manila. It utilizes 360-degree images, 2D images, voice recordings, and virtual reality technology to offer an immersive user experience of the virtual environment. The virtual tour system was developed using the Unity3D software and was then integrated into web and mobile applications accessible through web browsers and android smartphones, respectively.

Endemic Bird Image Classifier

This project uses deep learning to classify an image of endemic birds based on their extinction threat level. The data used in this image classifier is based on data sets gather from endemic birds that is located in the Philippines
